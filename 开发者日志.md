### 宿主机侧架构 v1.0

#### 技术栈
1. 窗口+UI：eframe(egui) 原生串口，自带标题栏和鼠标操作
2. 调试器驱动：probe-rs
3. 异步运行时：tokio RTT轮询线程+UI线程分离
4. 协议解析 自定协议 `"arm_targetheigh=500.0\n"`
5. 打包 `cargo build --release` 

#### 项目结构
``` rust
RTT_DebugTool/
└── rtt_debug_tool/
    ├── Cargo.toml
    └── src/
        ├── main.rs           # CLI → 连接 probe → 启动 egui App
        ├── app.rs            # egui App trait 实现 (主 UI 逻辑)
        ├── rtt_io.rs         # probe-rs RTT 通道读写 (后台线程)
        ├── watch_state.rs    # 变量树 (Arc<RwLock<>>) + 协议解析
        └── watch_widget.rs   # 自定义 Watch Tree 控件
```

#### 交互设计

1. 鼠标
    点击 ▸/▾ 图标：折叠或展开结构体
    点击 value 单元格：进入编辑模式，文本变为可输入框
    在编辑框中 Enter：确认修改 → `set arm.target_height 600.0\n` → RTT down
    在编辑框中 Esc：取消修改, 恢复原值
    点击窗口 ✕：关闭程序
    菜单栏 文件 → 退出：关闭程序

2. 键盘
    r：刷新，若窗口有焦点
    Ctrl+C：退出
    ↑/↓：在变量列表中移动光标
    Enter (在行上)：进入该变量的编辑模式
    Esc：取消编辑 / 关闭弹窗

#### 核心数据结构
```rust
// ============================================================
// watch_state.rs — 变量树
// ============================================================

#[derive(Clone)]
struct VarInfo {
    path:       String,       // "chassis.left_rpm"
    name:       String,       // "left_rpm"
    value:      String,       // "1000.5"
    type_name:  String,       // "f32"
    access:     Access,       // ReadWrite | ReadOnly
    children:   Vec<VarInfo>,  // 子字段 (结构体展开)
}

enum Access { ReadWrite, ReadOnly }

struct WatchState {
    roots: Vec<VarInfo>,      // 顶级条目
    last_update: Instant,
}

```
`VarInfo` 树由RTT Reader线程朱行更新：收到`"chassis.left_rpm=1000.5"` → 查树路径 → 替换 `value`。UI 线程只读 `Arc<WatchState>` 快照。

##### watch_widget.rs — 树形控件
```rust
impl RttWatchApp {
    fn show_var_row(&mut self, ui: &mut egui::Ui, var: &VarInfo, state: &WatchState) {
        ui.horizontal(|ui| {
            let indent = var.depth() * 20.0;

            if var.has_children() {
                // ▸ / ▾ 折叠按钮
                let collapsed = self.is_collapsed(&var.path);
                let arrow = if collapsed { "▸" } else { "▾" };
                if ui.button(arrow).clicked() {
                    self.toggle_collapse(&var.path);
                }
                ui.label(&var.name);       // 结构体名
                ui.label("[Struct]");
            } else {
                // 普通变量行
                ui.add_space(indent);       // 缩进
                ui.label(&var.path);        // 路径

                // 值单元格 ——— 核心交互
                ui.with_layout(egui::Layout::right_to_left(egui::Align::Center), |ui| {
                    match self.editing.as_ref() {
                        // 编辑模式: 显示文本输入框
                        Some(edit) if edit.path == var.path => {
                            let resp = ui.add(
                                egui::TextEdit::singleline(&mut self.edit_buffer)
                                    .desired_width(100.0)
                            );
                            if resp.lost_focus() || ui.input(|i| i.key_pressed(egui::Key::Enter)) {
                                self.commit_edit();   // 发送 set 命令
                            }
                            if ui.input(|i| i.key_pressed(egui::Key::Escape)) {
                                self.cancel_edit();
                            }
                        }
                        // 显示模式: 显示值, 点击进入编辑
                        _ => {
                            let value_text = if var.access == Access::ReadWrite {
                                egui::RichText::new(&var.value).color(egui::Color32::WHITE)
                            } else {
                                egui::RichText::new(&var.value).color(egui::Color32::GRAY)
                            };
                            let resp = ui.add(
                                egui::Button::new(value_text)
                                    .fill(egui::Color32::from_gray(32))
                                    .min_size(egui::vec2(100.0, 20.0))
                            );
                            if resp.clicked() && var.access == Access::ReadWrite {
                                self.start_edit(&var.path, &var.value);
                            }
                        }
                    }
                });

                // 访问权限标签
                let access_color = match var.access {
                    Access::ReadWrite => egui::Color32::GREEN,
                    Access::ReadOnly  => egui::Color32::GRAY,
                };
                ui.label(egui::RichText::new(format!("{:?}", var.access)).color(access_color));
            }
        });

        // 递归子条目 (展开时)
        if !self.is_collapsed(&var.path) {
            for child in &var.children {
                self.show_var_row(ui, child, state);
            }
        }
    }
}
```

##### rtt_io.rs — 后台 RTT 读写
```rust
// 伪代码
struct RttChannel {
    tx: Sender<String>,  // 下行发送
    rx: Receiver<String>, // 上行接收
}

impl RttChannel {
    /// 连接 probe → attach → 定位 RTT up1/down0 → 启动后台线程
    fn connect(chip: &str, probe_speed: u32) -> Result<Self, Error> {
        let probe = Probe::list_all()?[0].open()?;
        let session = probe.attach(chip)?;
        let rtt = session.setup_rtt()?;

        let up   = rtt.up_channel(1)?;
        let down = rtt.down_channel(0)?;

        let (tx, _) = channel();
        let (_, rx) = channel();

        // 上行 Reader 线程: 持续读 up1 → 解析行 → 更新 WatchState
        thread::spawn(move || {
            let mut buf = String::new();
            loop {
                let bytes = up.read(1024);
                buf.push_str(&String::from_utf8_lossy(&bytes));
                // 按 \n 拆分成行, 交 state 更新
                while let Some(pos) = buf.find('\n') {
                    let line = buf[..pos].to_string();
                    update_watch_state(&line);
                    buf = buf[pos + 1..].to_string();
                }
            }
        });

        Ok(Self { tx, rx })
    }

    /// PC → MCU: 发送 set 命令
    fn send_cmd(&self, cmd: &str) {
        let frame = format!("{}\n", cmd);
        self.down.write(frame.as_bytes());
    }
}
```
